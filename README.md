<!-- PROJECT DESCRIPTION -->
<h2>📌 Project Description</h2>

<p>
<b>SafeCampus</b> is a campus safety tracking platform that allows students to report safety issues,
track complaints, and enables administrators to monitor and resolve them efficiently.
</p>

<p>
This system improves safety by providing a centralized complaint reporting and management interface.
</p>

<ul>
<li>Report campus safety issues</li>
<li>Track complaint status in real-time</li>
<li>Admin dashboard to manage complaints</li>
<li>Emergency SOS interface</li>
</ul>

<!-- INSTALLATION -->
<h2>⚙ Installation</h2>

<h3>Method 1: Run Locally</h3>

<pre>
git clone https://github.com/yourusername/safecampus.git
cd safecampus
</pre>

<p>Open <b>index.html</b> in browser</p>

<h3>Method 2: Using VS Code</h3>

<ol>
<li>Open project folder</li>
<li>Install Live Server extension</li>
<li>Right click index.html</li>
<li>Select Open with Live Server</li>
</ol>
